# Code of Conduct

## Purpose
The purpose of this Code of Conduct is to ensure that this project remains a welcoming, respectful, and 
collaborative environment for all contributors and users. We value diversity and inclusion and expect 
all participants to behave professionally and courteously.

## Expected Behavior
All contributors are expected to:
* Treat others with respect, courtesy, and empathy
* Focus feedback on ideas, code, and processes, not individuals
* Be open to constructive discussions and differing opinions
* Actively contribute to a positive and collaborative environment
* Respect intellectual property rights

## Unacceptable Behavior
The following behaviours are not tolerated:
* Harassment, intimidation, or discriminatory language or imagery
* Personal attacks, insults, or public shaming
* Disrespecting others' privacy or contributions
* Any behavior deemed unprofessional or inappropriate in a collaborative environment

## Reporting
If you experience or witness unacceptable behavior that violates this Code of Conduct, please contact

## Enforcement
Contributors who violate this Code of Conduct may receive warnings or have their contributions removed. 
Serious or repeated violations may result in temporary or permanent exclusion from the project.
